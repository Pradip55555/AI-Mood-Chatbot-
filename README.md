# 🤖 AI Mood Chatbot

An advanced AI-powered chatbot built using Streamlit, LangChain, and Mistral AI with multiple personalities and a modern futuristic UI.

The chatbot can switch between different moods like Angry 😡, Funny 😂, and Sad 😢 while maintaining conversation memory in real time.

---

# 🚀 Features

✅ Multiple AI personalities
✅ Modern futuristic UI
✅ Session memory support
✅ Real-time AI responses
✅ Chat history download
✅ Interactive sidebar controls
✅ Adjustable AI creativity
✅ Responsive Streamlit design

---

# 🎭 AI Personalities

The chatbot supports multiple moods:

* 😡 Angry Mode
* 😂 Funny Mode
* 😢 Sad Mode

Each personality changes the AI response style dynamically.

# ⚙ Installation

## 1️⃣ Clone Repository

```bash id="l6r7xz"
git clone https://github.com/YOUR_USERNAME/AI-Mood-Chatbot.git
```

---

## 2️⃣ Navigate to Project

```bash id="e8z80j"
cd AI-Mood-Chatbot
```

---

## 3️⃣ Create Virtual Environment

```bash id="m0r4xr"
python -m venv .venv
```

Activate virtual environment:

### Windows

```bash id="u0fg2f"
.venv\Scripts\activate
```



## 4️⃣ Install Dependencies

```bash id="mctdd5"
pip install -r requirements.txt
```

Or install manually:

```bash id="wtx4u2"
pip install streamlit langchain langchain-mistralai python-dotenv
```

---

# 🔑 Environment Variables

Create a `.env` file in the root directory:


https://console.mistral.ai/

# 🧠 How It Works

1. User selects chatbot personality
2. User sends a message
3. LangChain manages conversation memory
4. Mistral AI generates response
5. Streamlit displays styled chat UI
